## 공문서 작성 규칙 요약

### 기본 구조

1. 두문: 기관명, 수신, 경유, 제목 등 문서의 식별 정보
2. 본문: 관련 근거, 목적, 세부 내용, 붙임
3. 결문: 발신명의, 결재선, 시행 및 접수 정보, 연락처, 공개구분

### 표기 규칙

- 숫자는 아라비아 숫자를 사용합니다.
- 날짜는 `2026. 4. 3.`처럼 연, 월, 일을 마침표로 구분하고 앞자리 `0`은 쓰지 않습니다.
- 시간은 `09:00`처럼 24시간제로 씁니다.
- 금액은 `금123,456원(금일이삼사오육원)`처럼 숫자와 한글 표기를 함께 씁니다.
- 여러 쪽 문서는 최종 출력 시 쪽 번호를 넣습니다.

### 항목 번호 체계

`1. → 가. → 1) → 가) → (1) → (가)`

문서 한 건 안에서는 위 순서를 유지하면 읽기와 결재가 쉬워집니다.

## 자주 하는 실수

- 제목을 문장처럼 길게 작성하는 경우
- `관련:` 근거나 목적 문장을 빼고 바로 본론으로 들어가는 경우
- 붙임 수량과 본문 설명이 서로 맞지 않는 경우
- 공개구분, 시행번호, 접수란을 비워 둔 채 대외 시행하는 경우
- 민원 회신문에서 이의신청 안내를 누락하는 경우

## 작성 팁

- `{중괄호}`는 실제 값으로 바꿉니다.
- `<!-- GUIDE: -->` 주석은 작성 안내이므로 최종 배포 전 삭제합니다.
- 표 안의 빈 항목은 기관 실무에 맞게 줄이거나 늘릴 수 있습니다.
- 전자문서시스템에 옮겨 적을 때는 기관 내부 규정을 한 번 더 확인합니다.
