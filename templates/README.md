### 사용 방법

1. 필요한 템플릿 파일을 MDEditor에서 엽니다.
2. `{중괄호}` 안의 내용을 실제 값으로 교체합니다.
3. `<!-- GUIDE: -->` 주석은 작성 안내이며, 완성 후 삭제합니다.
4. `.md`로 저장하거나 PDF로 내보냅니다.

### 참고

- 이 템플릿은 공문서 작성의 편의를 위한 참고용 양식입니다.
- 법적 효력이 있는 공식 서식을 대체하지 않습니다.
- 실제 공문서는 각 기관의 전자문서시스템(온나라 등)을 통해 시행합니다.
